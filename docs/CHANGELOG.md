## 2026-06-20 — 모바일 반응형 보강 (결과 모달 · 삭제 버튼)

- 결과 모달이 뷰포트를 넘지 않도록 `max-height`(dvh) + 내부 스크롤 적용. 헤더와 `돌아가기` 버튼은 항상 보임 (`index.html`).
- 모바일 가로에서 순위 목록을 **2열 그리드**로 배치(모달 폭 확대)해 6명도 스크롤 없이 표시.
- 참가자 행 `input`에 `min-width:0`을 적용해 `✕` 삭제 버튼이 카드 밖으로 넘치던 문제 수정. 버튼을 원형으로 다듬음.
- 서비스워커 캐시 `v10` → `v11`.
- 회귀 테스트 2종 추가(모바일 세로 삭제버튼 / 모바일 가로 결과모달) — 전체 **8종 통과**.
- 에이전트: Claude (spec-driven-development 6 Gates).
